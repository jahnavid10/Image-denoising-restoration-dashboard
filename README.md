# Image Denoising and Restoration Dashboard

This project is a simple image denoising and restoration dashboard built using **Gradio**, **TensorFlow/Keras**, and **OpenCV**.

Users can upload a noisy, brightened, or darkened image and restore it using trained deep learning models.

## Live Demo

Hugging Face Space:  
`https://huggingface.co/spaces/dulalajahnavi10/Image-Denoising-Dashboard`

## Models Used

- Autoencoder
- CBDNet
- IRCNN
- U-Net
- RCAN

## Features

- Upload noisy images
- Restore images using one selected model
- Compare outputs from all models
- Display image quality metrics
- Show inference time

## Recommended Input

The models were trained on **256 × 256 RGB images**.

For best results, upload images with size:

```text
256 × 256 pixels
```

Images with other sizes are resized to **256 × 256** before prediction.

## Metrics

The dashboard shows no-reference image metrics:

- Brightness
- Contrast
- Sharpness
- Entropy
- Underexposed percentage
- Overexposed percentage
- Inference time

Since no clean ground-truth image is uploaded, full-reference metrics such as **PSNR**, **SSIM**, **MSE**, and **MAE** are not calculated.

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/image-denoising-restoration-dashboard.git
cd image-denoising-restoration-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Download the model files:

```bash
python download_models.py
```

Run the app:

```bash
python app.py
```

Open the local Gradio URL shown in the terminal.
